This App is written in Kotlin and base on MVVM material design uses
the following Components:
 - ViewModel
 - LiveData
 - View Binding
 - Navigation
 - Room
 - Retrofit
 - Paging
 - RXJava
 - Viewpager2
 - EventBus
 - Fresco
 - Material design

It uses Koin for dependency injection.
